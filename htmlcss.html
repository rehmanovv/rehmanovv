HTML & CSS Dərsləri

Web dünnyasına ilk addım buradan başlayır!

HTML nədir?

HTML (Hypertext Markup Language) veb səhifələrin strukturunu qurmaq ücün istifade olunan əsas dildir. Bütün saytların əsas gövədəsi HTML-dən ibarətdir.

CSS nədir?

CSS (Cascading Style Sheets) veb səhifələrə stil vermək ücün istifadə olunur. Saytın gözəlliyinə, dizayn elementlərinə cavabdehdir.

Niyə HTML & CSS öyrənməlisən?

Əgər frontend proqramçı olmaq, öz saytlarını qura bilmək istəyirsənsə, HTML və CSS ilk addımındır. Onlar olmadan veb dünnyasına daxil olmaq imkansızdır.
